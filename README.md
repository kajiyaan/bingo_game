# 🎉 パーティゲームズ

スマホでQRを読むだけで、みんなで遊べるパーティゲーム集です。準備ゼロ・アプリのインストール不要。宴会・研修・懇親会にどうぞ。

**今あるゲーム**
- 🎱 ビンゴ大会 — ルーレット抽選・スマホカード・達成順位
- 🤝 以心伝心 — お題に回答、多数派と一致でポイント
- ⚡ 早押しクイズ — 4択＋速く答えるほど高得点

## つかう（別のPCで動かす）

### 必要なもの
- [Node.js](https://nodejs.org/)（バージョン18以上）
- [Git](https://git-scm.com/)

### 手順
```bash
git clone https://github.com/kajiyaan/bingo_game.git
cd bingo_game
npm install
node server.js
```
ブラウザで **http://localhost:3000** を開くと、ゲーム選択画面が出ます。

> 同じWi-Fiのスマホから参加者としてつなぐ場合は、司会PCのIPアドレスを使って
> `BASE_URL=http://<司会PCのIP>:3000 node server.js` のように起動すると、QRが正しいURLを指します。

## つくる（ワークショップ向け）

このプロジェクトは「**新しいゲームを足しやすい土台**」でできています。プログラミング未経験でも、**Claude Code に相談しながら**新しいゲームを1本作れます。

大まかな流れ：
1. このリポジトリを clone して、フォルダを Claude Code で開く
2. Claude に「◯◯というゲームを追加したい」と伝える
3. Claude がコードを書く → ブラウザで動かして確認 → 対話で直していく
4. 動いたら完成！

Claude Code は、リポジトリ内の [`CLAUDE.md`](CLAUDE.md) を読んで、このプロジェクトの作り方（土台とゲームの分け方・追加手順）を理解した上で手伝ってくれます。

### フォルダの地図（ざっくり）
```
server.js          … 全体の入り口
lib/               … 全ゲーム共通の仕組み（部屋の管理など）
games/<ゲーム名>/    … 各ゲームの中身（ルールや進行）
public/            … 画面（ランチャー・司会画面・スマホ画面）
```
1つのゲーム＝ `games/` の中の1フォルダ ＋ `public/` の中の画面、でできています。
